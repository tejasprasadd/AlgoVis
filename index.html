Bubble Sort

Compares adjacent elements in an array and exchanges their positions based on their values. The name comes from the way it looks, similar to bubbles floating and sinking in water.

Selection Sort

A straightforward algorithm that's a good option for smaller data sets, but less efficient for larger ones.

Insertion Sort

Places an unsorted element in its appropriate position in each iteration, similar to sorting cards in a game

Quick Sort

Uses a divide-and-conquer method to organize data by selecting a pivot, partitioning the array around it, and recursively sorting the partitions. It's usually faster than merge sort.

Visualize Quick Sort
Merge Sort

Uses the divide and conquer approach, splitting an array into two equal halves until each half has only one element. Then, the halves are recursively merged until the array is sorted
